Save

Saves data to a file.

Channels

Inputs

Examples (ExampleTable)
Attribute-valued data set.

Outputs

(None)

Description

Takes a data set provided at the input channel and saves it to a data file. The widget remembers a list of file names most recently used. It supports manual mode (one needs to press Save button to write the file) or automatic mode where any data that comes to widget's is immediately written to the selected file. Since the latest overrider the previous file withouth warning, manual mode is a default one.
